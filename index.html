Why Git & Open Source?

What Is Version Control?

Think of it like a Video Game "Save Point".

  • If you make a mistake in code, you can "reload" the previous save.
  • It tracks every change: Who changed it, When, and Why.

Why Industry Loves Open Source

Meta (Facebook)

Google

Why it matters for you: Contributing to these projects proves your skills to employers better than a resume.

Installation & Setup

Before we code, we must install the tools.

1. Install Git

  • Windows: Download "Git Bash" from git-scm.com.
  • Mac: Type git --version in Terminal (triggers install) or use Homebrew.
  • Linux: sudo apt-get install git

2. Configure Identity

Tell Git who you are (this appears in your history):

git config --global user.name "Your Name" git config --global user.email "youremail@example.com"

The Git Workflow

This is the heartbeat of modern software development. Memorize this cycle.

Working Dir
(Where you code)
→
Staging Area
(Preparing files)
→
Repository
(Saved History)
→
Remote (GitHub)
(The Cloud)

1. Initialize & Add

Start tracking a folder and prepare files to be saved.

# Start a new repo git init # Add specific file git add filename.py # Add ALL files git add .

2. Commit (The Save)

Permanently save the changes in Staging to history.

git commit -m "Added login feature"

3. Push & Pull

Syncing with the cloud (GitHub).

  • Push: Send your commits UP to GitHub.
  • Pull: Bring changes DOWN from GitHub to your computer.
git push origin main git pull origin main

4. Git Clone

Used when you want to download an existing project from GitHub to start working on it.

git clone https://github.com/username/projectname.git

Your GitHub Profile

Your profile is your new resume. Here is how to make it stand out:

  • Profile Pic: clear, professional, or a consistent avatar.
  • Bio: Short and sweet. (e.g., "Full Stack Dev | Open Source Enthusiast").
  • The Special Repo: Create a repository named exactly the same as your username (e.g., username/username). The README from this repo appears on your profile front page!
  • Green Squares: Code consistently to fill up your contribution graph.

Summary & Next Steps
